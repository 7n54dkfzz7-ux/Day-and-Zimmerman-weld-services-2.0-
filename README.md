# 🏗️ Weld Services - Recruitment Quiz

A fun, interactive recruitment form for Day and Zimmerman's Weld Services. This is a multi-page form designed to collect information about potential crew members.

## 🎯 Features

- **4-Page Interactive Form** with progress tracking
- **Form Sections:**
  - **Page 1:** Personal information (name, favorite tool, dream project)
  - **Page 2:** Safety knowledge and PPE practices
  - **Page 3:** Crew skills and work style preferences
  - **Page 4:** Commitment level and additional comments

- **Form Validation** - All fields must be completed before proceeding
- **Loading Animation** during form submission
- **Fun Surprise Ending** - A rickroll for all who complete the form!

## 🚀 How to Use

1. Open `index.html` in your web browser
2. Fill out all fields on each page
3. Navigate through pages using Back/Next buttons
4. Submit the final form to see your result!

## 🎨 Design Features

- **Dark Theme** - Professional construction aesthetic (#20242a background)
- **Orange Accents** - Energy and safety-focused branding (#ff8c00)
- **Responsive Design** - Works on desktop and mobile devices
- **Progress Bar** - Visual indicator of form completion
- **Smooth Transitions** - User-friendly navigation

## 📝 Form Data

All form responses are collected in the browser's `answers` object. You can expand the functionality to:
- Send data to a backend server
- Store in a database
- Generate reports
- Send confirmation emails

## 🎉 Current Endpoint

The form currently ends with a surprise rickroll when users click "Watch Your Result".
